## Design System Standards

### Color Palette

**Primary Colors:**
- **Teal-600**: `#0d9488` - Primary actions, links, highlights
- **Teal-700**: `#0f766e` - Hover states
- **Teal-500**: `#14b8a6` - Focus states

**Secondary Colors:**
- **Slate-900**: `#0f172a` - Text, headers
- **Slate-600**: `#475569` - Secondary text
- **Slate-100**: `#f1f5f9` - Backgrounds

**Status Colors:**
- **Success**: Emerald-600 `#059669`
- **Warning**: Amber-600 `#d97706`
- **Error**: Rose-600 `#dc2626`
- **Info**: Blue-600 `#2563eb`

### Typography

- **Font Family**: Instrument Sans (Primary), System fonts (Fallback)
- **Headings**: 
  - H1: `text-2xl font-bold`
  - H2: `text-xl font-semibold`
  - H3: `text-lg font-semibold`
- **Body**: `text-sm` or `text-base`
- **Small Text**: `text-xs`

### Spacing System

Based on Tailwind CSS spacing scale:
- **xs**: `0.5rem` (8px)
- **sm**: `0.75rem` (12px)
- **md**: `1rem` (16px)
- **lg**: `1.5rem` (24px)
- **xl**: `2rem` (32px)

### Icon System

- **Library**: Lucide React
- **Size Standards**:
  - Small: `w-4 h-4` (16px)
  - Medium: `w-5 h-5` (20px)
  - Large: `w-6 h-6` (24px)

### Component Standards

1. **Buttons**
   - Primary: `bg-teal-600 text-white hover:bg-teal-700`
   - Secondary: `bg-slate-200 text-slate-900 hover:bg-slate-300`
   - Danger: `bg-rose-600 text-white hover:bg-rose-700`

2. **Forms**
   - Input: `border border-slate-300 rounded-lg focus:ring-2 focus:ring-teal-500`
   - Label: `text-sm font-medium text-slate-700`

3. **Cards**
   - Container: `bg-white rounded-lg border border-slate-200 shadow-sm`
   - Padding: `p-5` or `p-6`

4. **Tables**
   - Header: `bg-slate-50 text-slate-900 font-semibold`
   - Row: `hover:bg-slate-50 transition-colors`
